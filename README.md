## Project Description
Yeh ek modern aur interactive developer portfolio hai jo Deadpool theme par based hai. Is project mein GSAP (GreenSock Animation Platform) , glsl aur Three.js ka istemal kiya gaya hai taaki ek engaging aur visually appealing user experience create kiya ja sake.


## Technologies Used
- **HTML5**: Structure ke liye.
- **CSS3**: Styling aur animations ke liye.
- **JavaScript**: Functionality aur interactivity ke liye.
- **GSAP**: Smooth animations ke liye.
- **Three.js**: 3D graphics aur effects ke liye.

## How to Run
1. **Dependencies Install Karein**:
   ```bash
   npm install
   ```
2. **Development Server Start Karein**:
   ```bash
   npm run dev
   ```
3. Browser mein open karein: `http://localhost:5173`
